# v0.1.0 Changelog

Status: historical release notes for the first Opstage CE public preview.

## Theme

Initial single-workspace control plane for Xtrape Capsule CE.

## Added

- Fastify backend with session authentication, CSRF protection, audit logging, and static UI hosting.
- React/Ant Design admin console for Dashboard, Users, Agents, Capsule Services, Commands, Audit Events, and Settings.
- SQLite runtime persistence with workspace, user, token, agent, service, config, action, command, command result, and audit tables.
- Agent registration tokens, hashed Agent tokens, heartbeat, service report, command dispatch, command result reporting, command cancellation, and expiration.
- RBAC roles: owner, operator, viewer.
- Diagnostics, metrics, audit export, SQLite backup, Dockerfile, Compose example, healthcheck, and deployment runbook.

## Security

- Password hashing with bcryptjs, HTTP-only signed session cookie, CSRF token requirement for admin mutations, hash-only Agent tokens, and secret redaction.

## Known limitations

- Single-workspace-first CE runtime.
- SQLite-backed local/private deployment target.
